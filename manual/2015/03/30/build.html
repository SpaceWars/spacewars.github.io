Instalando dependências

Para a compilação da aplicação é necessário ter instalado em seu computador os pacotes listados no arquivo requirements.txt. Por ser uma aplicação predominantemente escrita em Python, é possível instalar todas as dependências utilizando o easy_install ou o pip.

$ pip install -r requirements.txt

Iniciando aplicação

Instalados os pacotes, basta executar o arquivo main.py para que a aplicação seja inicializada.
